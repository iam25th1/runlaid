# Workplace Zones

5 zones, triggered at progress thresholds: 0%, 20%, 40%, 60%, 75%

Zone transitions swap background/furniture/items but keep all characters in place.

## Zone 0: Tech Office (0–20%)
- **Sky:** #0c1015 | **Accent:** #3fb950 (green)
- **Floor:** Grid pattern (tech/server room feel)
- **Desks:** 5 stations with monitors (screen glow in green)
- **Items:** CODE, PR, BUG FIX, API, DEPLOY, TEST
- **Icons:** code files, documents, folders

## Zone 1: Design Studio (20–40%)
- **Sky:** #14100a | **Accent:** #E8A000 (amber)
- **Floor:** Wood plank lines
- **Desks:** Mix of easels (with canvas + legs) and tablets
- **Items:** WIREFRAME, LOGO, MOCKUP, LAYOUT, ICON, PALETTE
- **Icons:** documents, images, folders

## Zone 2: Newsroom (40–60%)
- **Sky:** #080808 | **Accent:** #E24B4A (red)
- **Floor:** Tile grid
- **Desks:** Standard desks with monitors
- **Items:** ARTICLE, HEADLINE, SCOOP, REPORT, PHOTO, EDIT
- **Icons:** documents, images, folders

## Zone 3: Film Set (60–75%)
- **Sky:** #150008 | **Accent:** #FFD700 (gold)
- **Floor:** Carpet checkerboard pattern
- **Desks:** Cameras on tripods, director chairs, desks
- **Items:** SCRIPT, SCENE, CUT, VFX, TAKE, EDIT
- **Icons:** documents, folders, code files

## Zone 4: Trading Floor (75–100%)
- **Sky:** #040a04 | **Accent:** #E8A000 (amber)
- **Floor:** Marble with gold accent lines
- **Desks:** Multi-screen terminals and desks
- **Items:** REPORT, FORECAST, TRADE, MODEL, AUDIT, BRIEF
- **Icons:** documents, charts, code files, folders
