Privacy Policy

RUNLAID.exe — The Great Displacement

Last updated: March 28, 2026

Summary

RUNLAID.exe collects no data. Period.

Data Collection

This extension does not collect, store, transmit, or process any user data of any kind. Specifically:

  • No personal information is collected
  • No browsing history is accessed or stored
  • No conversation data from Claude is read, stored, or transmitted
  • No analytics or usage tracking of any kind
  • No cookies are set
  • No third-party services are contacted

Network Requests

RUNLAID.exe makes zero network requests. The extension runs entirely locally in your browser. No data is sent to any server, API, or third-party service.

Permissions

The extension requires only one permission:

  • Content script on claude.ai — Used solely to detect when Claude is streaming a response (by checking for a DOM element) and to inject the game canvas into the page. The extension does not read, modify, or interact with your conversations in any way.

Local Storage

The extension does not use localStorage, sessionStorage, chrome.storage, IndexedDB, or any other form of persistent storage. Game state exists only in memory during gameplay and is discarded when the overlay closes.

Third-Party Sharing

No data is shared with any third party because no data is collected.

Open Source

RUNLAID.exe is fully open source under the GPL v3 license. You can verify all of the above by inspecting the source code at github.com/iam25th1/runlaid.

Share Feature

The "Share to X" feature generates a PNG image locally in your browser and opens a Twitter/X intent URL. The PNG is created on your device — no image is uploaded to any server. The Twitter intent URL contains only the pre-filled tweet text, which you can review and edit before posting.

Changes

If this privacy policy changes, the updated version will be posted at this URL and the extension listing will be updated accordingly.

Contact

For questions about this privacy policy, open an issue at github.com/iam25th1/runlaid/issues or reach out on X at @25thprmr.
